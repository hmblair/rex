# rex — Remote Execution CLI

Use `rex` to run anything requiring significant compute or GPU resources on a remote host.

## Before running

- Host aliases are defined in `~/.config/rex/config.toml`. Read it to find available hosts.
- Run `rex --connection` to see which hosts have active connections. If there is no open connection, prompt the user to open one.

For full usage, commands, configuration, and examples, see [README.md](./README.md).
